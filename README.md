# Sparkle Squad Giveaway

Next.js App Router + TypeScript app for public lead capture and admin prize wheel, backed by Supabase Postgres/Auth.

## Quick start

1. Copy `.env.example` to `.env.local` and fill values.
2. Run `npm install`.
3. Run `npm run dev`.
4. Apply SQL in `supabase/migrations/202602111430_init.sql` in Supabase SQL Editor.
5. Configure Supabase Google provider and redirect URLs (see setup section in assistant response).
